# Dataset Description

This study uses the French Enedis dataset, which provides half-hourly electricity consumption data for supply points ≤ 36 kVA connected to the Enedis network.

## Enedis Dataset Statistics
- Raw data total rows: 43,219,248
- After data cleaning, valid residential load data: 20,528,616 rows
- Total original user groups: 468
- Valid user groups (time series length > 40,000): 60 groups | Top 60 selected for the experiment
- Final experimental dataset after filtering: 2,635,064 rows

To obtain the dataset, please visit the official public channel:  
<https://opendata.enedis.fr/datasets/conso-inf36-region>
